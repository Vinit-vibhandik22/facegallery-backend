# Stack Reference

## Frontend
- **Framework**: `Next.js 16.1.6` (App Router)
- **Language**: `TypeScript 5`
- **UI Library**: `React 19.2.3`
- **Styling**: `Tailwind CSS 4.2.1` with `PostCSS`
- **Animations**: `Framer Motion 12.38.0`, `GSAP 3.14.2`
- **Icons**: `Lucide React 0.576.0`
- **Scrolling**: `Lenis 1.3.19`

## Backend AI Engine
- **Framework**: `FastAPI` (Python)
- **Face Analysis**: `DeepFace` (Facenet512 model)
- **Face Detection**: `MTCNN`
- **Computer Vision**: `OpenCV (opencv-python-headless)`
- **Runtime**: `uvicorn`

## Data & Infrastructure
- **BaaS**: `Supabase` (@supabase/supabase-js 2.98.0)
- **File Processing**: `python-multipart`

## Build Tools
- **Linter**: `ESLint 9`
- **Package Manager**: `npm`
