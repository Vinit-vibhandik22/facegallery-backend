# Plan: 02-01-PLAN.md — Refinement Engine Implementation

**Goal**: Implement a centroid-based post-processing merge step in the FastAPI backend to unify fragmented clusters.

**Wave**: 1 (Implementation)
**Depends on**: Phase 1
**Requirements**: CLUS-04, INTG-01
**Artifacts**: `python_backend/main.py`, `tmp/test_refinement.py`

**must_haves**:
- Clusters representing the same person are unified via centroid similarity.
- Different people remain in separate clusters.

## Tasks

### 1. Implement `refine_labels_by_centroids` Helper
**<read_first>**
- `python_backend/main.py`
- `src/lib/clustering.ts` (Lines 171-230 for reference)
**<action>**
Add a helper function `refine_labels_by_centroids(embeddings: np.ndarray, labels: np.ndarray, threshold: float = 0.35)` to `python_backend/main.py` that:
1. Calculates the **centroid** (mean) embedding for each cluster.
2. Identifies pairs of clusters whose centroids are closer than the `threshold` (Cosine Distance).
3. Merges these clusters and returns an updated label array.
**<acceptance_criteria>**
`python_backend/main.py` contains the `refine_labels_by_centroids` function.
